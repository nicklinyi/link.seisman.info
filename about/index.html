About the site

This site collects a lot of useful sites I found during my daily life and researches.

The whole site was designed from scratch by me, a scientist but not a good web designer, themed by the Bootstrap framework and beautiful icons from Font Awesome, powered by the fastest static site generator Hugo.

The site is served freely on GitHub, with source codes available to public.

Contributing

Everyone is welcome to contribute to this site. Contributions include but not limited to:

  • Add more useful links
  • Report/Fix broken links
  • Improve the web design
  • Any brilliant ideas

For any contributions, please open an issue or submit a pull request.
